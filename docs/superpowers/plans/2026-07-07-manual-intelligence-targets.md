# Manual Intelligence Targets Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Let users manually add competitor-specific monitoring targets, validate them before activation, and feed valid targets into the existing competitive intelligence pipeline.

**Architecture:** Manual targets enter as frontend pending candidates in the competitor detail drawer. Activation continues through the existing backend `createTarget` validation path, while the backend persists target provenance and analysis notes as nullable metadata on `intel_monitor_target`.

**Tech Stack:** Vue 3, TypeScript, Element Plus, SCSS, Spring Boot Java, MyBatis-Plus, MySQL migration SQL.

## Global Constraints

- Keep the workbench competitor-list driven; do not replace the current page layout.
- Use Element Plus and existing local patterns; do not add a new component library.
- Validation-failed targets must not become active monitor targets.
- `analysisNotes` is user context only; it is not evidence for reports.
- Manual targets must use the existing collection, baseline, diff, report, task, delete, and one-click collection flows.
- Backend fields must be nullable and backward-compatible with existing callers.

- [x] **Step 1: Add submit helper**

```ts
function handleSubmitManualTarget() {
  if (!detailCompetitor.value) {
    ElMessage.warning('先选择一个竞品');
    return;
  }
  const title = manualTargetForm.title.trim();
  const url = manualTargetForm.url.trim();
  if (!title || !url) {
    ElMessage.warning('请填写目标标题和 URL');
    return;
  }
  drafts.value.unshift({
    competitorId: detailCompetitor.value.id,
    competitorName: detailCompetitor.value.name,
    type: manualTargetForm.type,
    title,
    url,
    status: 'draft',
    confidence: 1,
    source: 'manual',
    analysisNotes: manualTargetForm.analysisNotes.trim() || undefined,
    selected: true,
    validationStatus: 'pending',
    validationMessage: '保存时验活；通过后才会进入 active 监控',
    collectStatus: 'idle',
  });
  manualTargetDialogVisible.value = false;
  resetManualTargetForm();
}
```
